# Mantra Tiled Server

https://www.mapeditor.org

https://github.com/mapeditor/tiled


## Installation

```bash
npm install
```

## Run Server

```bash
node server.js
```

## Query Tiles

```bash
GET /tiles?x=<X_COORD>&y=<Y_COORD>&width=<WIDTH>&height=<HEIGHT>
```

## Tests

```bash
npm test

```
